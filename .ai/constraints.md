# 项目约束和偏好

> AI必须遵守以下约束，除非用户明确修改。

## 核心约束

### 1. 只做手机端网页
- 不考虑电脑端适配
- 所有布局以手机屏幕为基准（375px-430px宽度）
- 不需要响应式断点（md:、lg: 等）
- 目标：将来可迁移到小程序

### 2. 技术栈
- React + Vite + Tailwind CSS
- 部署到 Vercel

### 3. 代码风格
- 使用 Tailwind 类名，不写单独 CSS 文件
- 组件尽量小，一个文件一个组件
- 中文注释

## 文档规则
- 文档放在 `.ai/` 目录
- 问题解决后必须记录到 `.ai/solutions/`
- 记录格式：问题描述 → 原因 → 解决步骤
