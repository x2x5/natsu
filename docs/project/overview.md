# 项目概述

## 项目目的
这是一个**学习用网站项目**，目的是帮助用户理解网站开发的基本概念，学会如何指挥AI写代码、排查bug。

## 技术栈
- **React** — 前端框架，用组件方式写网页
- **Tailwind CSS** — 样式框架，直接在标签上写类名控制样式
- **Vite** — 构建工具，开发时热更新，打包时优化
- **Vercel** — 部署平台
