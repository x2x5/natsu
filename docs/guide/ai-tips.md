# 如何更好地指挥AI

## 提需求时，说清楚三件事

1. **想要什么效果**：描述最终用户看到的样子
2. **放在哪里**：首页？导航栏？弹窗？
3. **有什么交互**：点击后跳转？弹出提示？提交表单？

## 好的例子 vs 差的例子

- ❌ "帮我改一下页面"
- ✅ "在首页顶部加一个导航栏，左边是logo，右边是三个链接：首页、关于、联系，用手机看时变成汉堡菜单"

- ❌ "加个按钮"
- ✅ "在页面中间加一个蓝色按钮，文字是'提交'，点击后弹出一个确认对话框"

## 遇到AI改错时

1. 告诉AI**具体哪里不对**
2. 提供**浏览器控制台的报错信息**（复制完整报错）
3. 说明**期望的行为**是什么

## 提供报错信息的格式

```
我做了XXX操作后，页面出现以下问题：

控制台报错：
（粘贴完整报错）

期望的结果是：XXX
实际的结果是：XXX
```

## 让AI改代码时的注意事项

- 一次只让AI改一个功能，不要一次改太多
- 改完后先测试，再继续下一个功能
- 如果改坏了，告诉AI"回到上一步"或者"撤销刚才的改动"
