# 常用命令速查

## 开发
```bash
npm run dev          # 启动开发服务器
```

## 构建
```bash
npm run build        # 打包生产版本
npm run preview      # 本地预览
```

## Git
```bash
git add .            # 暂存所有改动
git commit -m "描述"  # 提交
git push             # 推送到GitHub
```

## 部署
```bash
vercel --prod        # 部署到Vercel
```

## 安装依赖
```bash
npm install xxx      # 安装新包
npm install -D xxx   # 安装为开发依赖
```
