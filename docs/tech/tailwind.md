# Tailwind CSS 常用类名

## 布局
- `flex` — 弹性布局
- `grid` — 网格布局
- `block` / `inline-block` — 块级/行内块
- `hidden` — 隐藏

## 间距
- `p-4` — 内边距（padding）
- `px-4` — 左右内边距
- `py-4` — 上下内边距
- `m-4` — 外边距（margin）
- `gap-4` — 子元素间距

## 尺寸
- `w-full` — 宽度100%
- `w-screen` — 宽度等于屏幕
- `h-screen` — 高度等于屏幕
- `max-w-xl` — 最大宽度
- `min-h-0` — 最小高度

## 文字
- `text-sm` / `text-base` / `text-lg` / `text-xl` / `text-2xl` — 字体大小
- `font-normal` / `font-bold` — 粗细
- `text-center` / `text-left` / `text-right` — 对齐
- `text-white` / `text-black` / `text-gray-500` — 颜色

## 颜色
- `bg-blue-500` — 蓝色背景
- `bg-gray-100` — 浅灰背景
- `text-red-500` — 红色文字
- `border-gray-300` — 灰色边框

## 圆角和阴影
- `rounded` / `rounded-lg` / `rounded-full` — 圆角
- `shadow` / `shadow-lg` / `shadow-xl` — 阴影

## 响应式
- `sm:` — 小屏幕起效
- `md:` — 中等屏幕起效
- `lg:` — 大屏幕起效
- 例：`md:flex lg:w-1/2`

## 交互
- `hover:bg-blue-600` — 鼠标悬停时
- `active:scale-95` — 点击时
- `focus:ring-2` — 聚焦时
- `cursor-pointer` — 鼠标指针
