# Natsu 项目文档索引

> 本目录用于帮助AI理解项目结构和用户意图，同时作为项目记忆库。

## 目录结构

```
docs/
├── INDEX.md              # 本文件：文档索引，找什么来这里
├── project/              # 项目相关
│   ├── overview.md       # 项目概述：做什么的、技术栈、部署信息
│   └── structure.md      # 文件结构：每个文件的作用
├── tech/                 # 技术知识
│   ├── react-basics.md   # React 基础：组件、State、Props、JSX
│   ├── tailwind.md       # Tailwind CSS：常用类名速查
│   └── vite.md           # Vite：常用命令和配置
└── guide/                # 使用指南
    ├── commands.md       # 常用命令速查
    ├── debug.md          # 常见Bug排查
    └── ai-tips.md        # 如何更好地指挥AI
```

## 快速查找

| 我想知道... | 看哪个文件 |
|-------------|-----------|
| 这个项目是做什么的 | `project/overview.md` |
| 文件结构是什么样的 | `project/structure.md` |
| React 组件怎么写 | `tech/react-basics.md` |
| Tailwind 类名怎么用 | `tech/tailwind.md` |
| 常用命令有哪些 | `guide/commands.md` |
| 页面报错怎么排查 | `guide/debug.md` |
| 怎么给AI提需求 | `guide/ai-tips.md` |
